# Todo Web App

A simple and clean web application to manage daily tasks.
Users can create, complete, and delete todos, with automatic persistence in the browser.

---

## Features

* Add new todos
* Mark todos as completed
* Delete todos
* Persist todos using browser local storage
* Simple and responsive UI

## Installation

Clone the repository

```
git clone https://github.com/yourusername/todo-web-app.git
```

Navigate to the project folder

```
cd todo-web-app
```

Install dependencies

```
npm install
```

Run the development server

```
npm run dev
```

The app will start locally at:

```
http://localhost:5173
```

---

## Usage

1. Type a task in the input field.
2. Press **Enter** or click **Add**.
3. Mark tasks as completed using the checkbox.
4. Remove tasks using the delete button.

All tasks are automatically saved in your browser.
